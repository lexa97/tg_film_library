# Интеграция Prowlarr - Краткое резюме

## Что было реализовано

✅ Полная интеграция с Prowlarr для поиска и скачивания торрент-раздач фильмов и сериалов  
✅ Кнопка "📥 Скачать" в результатах поиска и в деталях фильма  
✅ Поиск раздач с фильтром по качеству (720p и выше)  
✅ Фильтрация по категориям (Movies и TV)  
✅ Сортировка результатов по количеству сидов  
✅ Отображение до 10 лучших раздач с информацией о разрешении, источнике и размере  
✅ Автоматическая отправка выбранной раздачи в торрент-клиент через Prowlarr  
✅ Уведомление об успешной отправке или ошибке

## Что нужно сделать для запуска

### 1. Обновите .env файл

Добавьте следующие строки в ваш `.env` файл:

```env
PROWLARR_URL=http://prowlarr:9696
PROWLARR_API_KEY=ваш_api_ключ_здесь
```

> **Примечание**: API ключ вы получите после первого запуска Prowlarr (см. шаг 3)

### 2. Перезапустите Docker Compose

```bash
docker-compose down
docker-compose up -d
```

Это запустит новый контейнер с Prowlarr.

### 3. Настройте Prowlarr

1. Откройте http://localhost:9696 в браузере
2. Пройдите первоначальную настройку
3. Перейдите в **Settings → General** и скопируйте **API Key**
4. Добавьте API ключ в `.env` файл (см. шаг 1)
5. Добавьте индексаторы (трекеры) в **Indexers → Add Indexer**
   - Рекомендуем: RuTor, RuTracker, Kinozal
6. Перезапустите бота: `docker-compose restart bot`

**Подробная инструкция**: см. файл `PROWLARR_SETUP.md`

### 4. Проверьте работу

1. Откройте бота в Telegram
2. Найдите фильм (например, "Интерстеллар")
3. Нажмите кнопку **🧲 Magnet**
4. Выберите раздачу из списка
5. Скопируйте magnet-ссылку

## Как это работает

### Поток работы:

```
Пользователь → нажимает "🧲 Magnet"
     ↓
Бот → формирует запрос "название + год"
     ↓
Prowlarr → ищет в настроенных трекерах
     ↓
Бот → фильтрует (только 1080p+), сортирует по seeds
     ↓
Пользователь → видит список раздач с кнопками
     ↓
Пользователь → выбирает раздачу
     ↓
Бот → отправляет magnet-ссылку
```

### Пример отображения раздачи:

```
[👥 150] 1080p · 10.5 GB · RuTracker
```

Где:
- `[👥 150]` - количество сидов
- `1080p` - разрешение видео
- `10.5 GB` - размер файла
- `RuTracker` - источник (индексатор)

## Технические детали

### Фильтрация качества

Бот автоматически фильтрует раздачи и оставляет только:
- 2160p (4K)
- 1080p
- 720p (HD)

Раздачи более низкого качества (480p, SD) исключаются из результатов.

### Фильтрация по категориям

Бот ищет только в категориях:
- **2000** - Movies (фильмы)
- **5000** - TV (сериалы)

Это исключает из результатов музыку, игры, софт и другой контент.

### Кэширование

Результаты поиска кэшируются в памяти бота для текущей сессии. Это позволяет:
- Быстро получать magnet-ссылки без повторных запросов
- Экономить запросы к Prowlarr API

### Обработка ошибок

Бот обрабатывает следующие ситуации:
- Prowlarr недоступен → показывает сообщение об ошибке
- Раздачи не найдены → предлагает попробовать другой фильм
- Неправильные данные → показывает понятное сообщение

## Безопасность

⚠️ **Важно**: 
- Не публикуйте файл `.env` в репозиторий
- Держите Prowlarr за файрволом или настройте аутентификацию
- Используйте безопасные пароли для приватных трекеров
